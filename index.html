Pixeldragon

 by Andreas Trawoeger

Pixeldragon is based on five docker images that are build with using docker-compose.

  1. redis Database for Celery Task Queue
  2. flower Celery monitoring tool
  3. ids Image Distribution Service
  4. iipsrv Internet Image Protocol Server
  5. nginx HTTP Proxy
